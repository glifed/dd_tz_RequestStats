# Тестовое: Request Stats

Привет. Нужно поднять учебное приложение на выданном Windows Server.

**У тебя есть:** доступ к серверу, `request-stats.war`, скрипт миграции `init.sql`.

Приложение считает запросы и пишет историю в SQL Server. После деплоя: `http://<host>:8080/request-stats/`.  
Стек: JDK 11+, Tomcat 9, SQL Server Express.

---

## Что сделать

**1. Развернуть**  
Tomcat и Express — как службы Windows. БД из миграции, war в Tomcat, в браузере должно работать обновление запросов: UI и запись в историю.

**2. Написать гайд**  
Так, чтобы коллега без тебя мог повторить стенд.

**3. Бэкапы**  
Выбери и реализуй стратегию для БД `RequestStats`. В гайде опиши решение и как откатываться. Восстановление нужно **реально** прогнать в базу *_restore и зафиксировать результат.

**4. Логи Tomcat**  
Реши вопрос с retention логов Tomcat на стенде и опиши в гайде, что сделал.

---

## Что сдать

Рабочий сервер, гайд, скрипты по бэкапам и логам Tomcat, отчёт что restore сделан (скриншот).
